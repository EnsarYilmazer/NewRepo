# İnteraktif Medya Kitaplığı

Bu proje, **ISE-201 Web Teknolojileri** dersi kapsamında geliştirilmiştir. Kullanıcıların film/dizi verilerine erişebildiği, arama yapabildiği ve favorilerini yönetebildiği bir Single Page Application (SPA) uygulamasıdır.

##    Kullanılan Teknolojiler
* **HTML5:** Anlamsal etiketler kullanılarak yapılandırıldı.
* **CSS3:** Flexbox ve Grid ile duyarlı (responsive) tasarım sağlandı.
* **JavaScript (ES6+):** Fetch API, Async/Await ve LocalStorage kullanıldı.

##    Özellikler
* **Dinamik Veri:** Veriler yerel bir JSON dosyasından asenkron olarak çekilir.
* **SPA Yapısı:** Sayfa yenilenmeden detay ekranına geçiş yapılır.
* **Favoriler:** Seçilen içerikler `localStorage` kullanılarak tarayıcıda saklanır.
* **Duyarlı Tasarım:** Mobil, tablet ve masaüstü cihazlarla uyumludur.

##    Kurulum
1. Bu depoyu klonlayın.
2. `index.html` dosyasını tarayıcınızda açın.
